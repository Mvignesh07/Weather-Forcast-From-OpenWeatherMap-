OpenWeatherMap URL

I used web service url in this project is given below,
https://api.openweathermap.org/data/2.5/weather?q={CITY NAME}&appid={API ID}
Example:
https://api.openweathermap.org/data/2.5/weather?q=Pune&appid=60bab3f0f3e6d69db68e049b3990b213
After loading, it will have the JSON formatted data containing an array of datails. Each array item contains the data's like latitude, longitude, country, city/state,temp,huminity, city id.
